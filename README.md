# Student Management System

A Java-based CLI application developed using Core Java concepts to perform CRUD operations on student records.
## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Search Student
## Technologies Used
- Java
- ArrayList
- Scanner
- OOP Concepts
## Files
- Student.java
- StudentManagementSystem.java
## Run Commands
javac Student.java
javac StudentManagementSystem.java
java StudentManagementSystem
